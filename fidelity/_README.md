This directory contains python code to simulate homodyne photocurrent analysis. We deal with **timing jitter**, **phase jitter**, **mode and marginal inference**, and **Wigner function reconstruction**.

Most of the code is explained in the markdown cells, but at a glance, the analysis presented here supports the simulation's effectiveness. The Wigner Function Reconstruction is currently being performed using the inverse Radon transform, but a Maximum Likelihood Reconstruction is currently in the works and will be implemented soon.
